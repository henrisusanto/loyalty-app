# plugins

```
src/plugins
├── ejs.plugin.ts
├── hmr.plugin.ts
├── readme.md
└── static.plugin.ts
```

- `ejs.plugin.ts` is used to render react application in production mode.
- `hmr.plugin.ts` is used to hot module replacement in development mode.
- `static.plugin.ts` is used to host JS file in public folder.
